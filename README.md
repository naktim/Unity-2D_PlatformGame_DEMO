
# Unity-2D_PlatformGame_DEMO

2020年冬，参照教程实现的Unity 2D Platform Game Demo

实现功能：  
1.跳跃  
2.释放炸弹，炸弹爆炸、移动  
3.生命值、受击、死亡  
4.怪物：普通攻击、技能、状态机实现巡逻、追击等
5.UI：血条、开始界面、暂停界面、结束界面  
6.系统功能：开始游戏、关卡切换、存档、暂停、重新开始等  
7.TileMap绘制地图
